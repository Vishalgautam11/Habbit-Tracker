# 📊 Excel Habit Tracker — Monthly Dashboard

A fully automated monthly habit tracking system built in Microsoft Excel. Track your daily routines, monitor weekly progress, and visualize your consistency through an interactive dashboard — all in one file.

> Originally built for personal use and gained traction on Instagram for its clean design and practical utility.

## ✨ Features

- **Daily Task Grid** — Track 13+ habits across all 31 days of the month
- **Week-wise Progress** — Auto-calculated W1–W5 completion percentages per habit
- **Total Monthly Progress** — Overall completion rate for each habit at a glance
- **Monthly Dashboard** with:
  - 📊 Horizontal bar chart — per-task monthly progress %
  - 🍩 Donut chart — weekly overall progress breakdown
  - 📈 Line chart — daily progress trend across the month
- **Conditional Formatting** — Color-coded cells for instant visual feedback
- **Zero Manual Calculation** — Everything updates automatically as you fill in your daily tasks

---


## 🚀 How to Use

1. **Download** the `.xlsx` file and open it in Microsoft Excel (2016 or later recommended)
2. **Edit the task names** in the Daily Task column (Column A) to match your habits
3. **Mark each day** — enter `1` for completed, `0` (or leave blank) for not completed
4. **Watch the dashboard update** automatically — progress bars, charts, and percentages recalculate in real time
5. **Duplicate the sheet** at the start of each new month and reset the daily entries
